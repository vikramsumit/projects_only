# Portfolio HTML Version

This project is a static personal portfolio built using plain HTML and CSS. It is a lightweight version of a professional portfolio that focuses on clear structure, navigation, and content presentation.

The design is suitable for demonstrating basic frontend development skills and for presenting personal information, projects, and contact details in a simple and elegant manner.

## Features

- Responsive static portfolio pages
- About, projects, blog, and contact sections
- Clean HTML/CSS structure
- Lightweight and easy to host

## Technologies Used

- Language: HTML, CSS
- Framework: None
- Database: None
- Libraries: None
- Tools: VS Code

## Folder Structure

```text
portfolio2.html/

```
