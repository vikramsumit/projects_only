# Mini Projects Collection

This folder groups several small programming experiments and practice projects covering web development, scripting, backend logic, and user-interface design. It reflects steady hands-on learning across multiple technologies and problem-solving styles.

The projects in this section are useful for showcasing variety, rapid prototyping ability, and a practical understanding of common development patterns such as forms, routing, servers, and simple automation.

## Features

- Collection of lightweight coding experiments
- Frontend, backend, and scripting examples
- Practice work for web interfaces and server-side logic
- Easy-to-browse portfolio section for small applications

## Technologies Used

- Languages: Python, JavaScript, HTML, CSS
- Frameworks: Express, basic web technologies
- Database: None
- Libraries: Varies by subproject
- Tools: VS Code, GitHub, Node.js

