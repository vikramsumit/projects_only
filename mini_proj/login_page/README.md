# Login and Signup Page

This project is a simple static web page that demonstrates a basic authentication-style interface with login and signup forms. It is a lightweight example of frontend design and form layout.

The project is suitable for showcasing clean UI structure, responsive layout ideas, and beginner-level frontend implementation using plain HTML and CSS.

## Features

- Login form UI
- Signup form UI
- Responsive layout
- Clean static design
- Beginner-friendly frontend example

## Technologies Used

- Language: HTML, CSS
- Framework: None
- Database: None
- Libraries: None
- Tools: VS Code

## Folder Structure

```text

```

## Usage

Open the HTML files in a browser to view the login and signup screens.
