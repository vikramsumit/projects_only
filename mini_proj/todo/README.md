# To-Do Static Project

This mini-project is a simple to-do application built as a static web example. It demonstrates front-end DOM handling, user interaction, and basic task management concepts in a compact format.

The project is useful for showing how a simple idea can be implemented quickly and how HTML, CSS, and JavaScript can work together to create an interactive experience.

## Features

- Add tasks
- Basic task interaction
- Simple and lightweight UI
- Practice project for frontend logic

## Technologies Used

- Language: JavaScript, HTML, CSS
- Framework: None
- Database: None
- Libraries: None
- Tools: VS Code
