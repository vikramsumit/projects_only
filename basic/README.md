# Basic Web Projects

This folder contains beginner-friendly web development exercises that showcase simple UI implementation, frontend fundamentals, and practical React learning. The projects here are ideal for demonstrating steady growth from basic HTML/CSS work to more structured React applications.

The collection highlights clean design, component-based thinking, and the ability to iterate on user interface ideas quickly.

## Projects

- To-Do App
- To-Do Enhanced App

## Technologies Used

- Language: JavaScript
- Framework: React, Vite, Tailwind CSS
- Database: None
- Tools: VS Code, GitHub

## Folder Structure

```text
basic/
├── To-Do/
├── To-do_enhanced/
└── README.md
```

## Installation

```bash
cd To-Do
npm install
npm run dev
```

## Usage

Open the relevant project folder and run the development server to interact with the app locally.
