# To-Do Enhanced App

An upgraded version of the basic to-do project with a more polished interface and improved user experience. This version is intended to showcase a stronger frontend design sense and more practical UI patterns.

[![React](https://img.shields.io/badge/React-18-61DAFB)](https://react.dev/)
[![Vite](https://img.shields.io/badge/Vite-5-646CFF)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-38B2AC)](https://tailwindcss.com/)

## Overview

This application builds on the simple task tracker concept by improving the user experience, layout, and component structure. It is a useful example of how a beginner project can evolve into a more portfolio-ready frontend experience.

## Features

- More refined task management flow
- Better visual hierarchy and styling
- Responsive design improvements
- Cleaner component organization

## Technologies Used

- Language: JavaScript
- Framework: React, Vite
- Styling: Tailwind CSS
- Tools: VS Code, GitHub, npm

## Installation

```bash
npm install
npm run dev
```

## Usage

Run the app locally and use it as a task tracker with a more polished UI than the basic version.
