# Netflix Clone UI

This project is a simple frontend clone of a streaming service landing page, built using static HTML and CSS. It focuses on layout design, visual styling, and responsive presentation.

The project is intended as a practice exercise for improving UI development skills and understanding how polished landing pages are structured.

## Features

- Responsive landing page layout
- Styled hero section and content cards
- Static frontend design exercise
- Portfolio-ready UI practice project

## Technologies Used

- Language: HTML, CSS
- Framework: None
- Database: None
- Libraries: None
- Tools: VS Code
