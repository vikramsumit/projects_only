# Portfolio Website

A modern personal portfolio built with Next.js and TypeScript. This project presents a refined, responsive, and professional online presence suitable for internships, placements, and freelance opportunities.

[![Next.js](https://img.shields.io/badge/Next.js-15-000000)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-38B2AC)](https://tailwindcss.com/)

## Overview

This version of the portfolio emphasizes a professional presentation with clean sections, responsive design, and a polished user experience. It is designed to be easy to customize and suitable for showcasing projects, experience, and skills.

## Features

- Responsive one-page portfolio experience
- Modern navigation and hero section
- Project and blog-friendly structure
- Dark/light theme support
- Clean styling with Tailwind CSS

## Technologies Used

- Language: TypeScript
- Framework: Next.js
- Styling: Tailwind CSS
- Animation: Framer Motion
- Tools: VS Code, GitHub, npm

## Installation

```bash
npm install
npm run dev
```

## Usage

Open the local development server to view the portfolio and customize the sections with your own details.
